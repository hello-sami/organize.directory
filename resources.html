Resources for Starting a Mutual Aid Network

Getting Started Guide

1. Assess Community Needs

  • Survey your community to identify gaps in support
  • Connect with existing community organizations
  • Document specific needs and resources available

2. Build Your Core Team

  • Recruit dedicated volunteers
  • Define roles and responsibilities
  • Establish communication channels
  • Create decision-making processes

3. Set Up Infrastructure

  • Choose communication tools (Signal, WhatsApp, etc.)
  • Set up resource tracking systems
  • Create intake forms and processes
  • Establish safety and security protocols
